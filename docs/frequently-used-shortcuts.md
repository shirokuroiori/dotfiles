# よく使うショートカット集

こちらによく使うショートカットを記載します。
利用者から頻繁に質問されるのでAIで回答できるように記載します。

## ジャンプリストを戻る

定義ジャンプした後にジャンプ前の元の場所に戻る方法

`<C-o>`

## ジャンプリストを進む

ジャンプリストを戻って、その逆をする

`<C-i>`

## LSPメッセージをコピーする

LSPから表示されているエラー内容をクリップボードにコピーする
`.config/nvim/lua/config/keymaps.lua`に定義されている

`<leader>dy`

## Weztermのペイン移動

`<C-S-hjkl>`

## Weztermのペインサイズ変更

`<M-S-hjkl>`

## ペイン分割

`<C-M-S>`

### 縦

+ `"`

### 横

+ `%`

### undoの逆

`<C-r>`

### 保存せずバッファを閉じる

`:bd! xx`

### ターミナルモードから抜ける

`<C-\\><C-n>`
